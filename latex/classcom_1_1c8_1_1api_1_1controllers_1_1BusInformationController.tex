\doxysubsection{Detailed Description}
REST controller for retrieving information about bus locations.

This controller handles requests for fetching bus data, filtering depoted buses, and optionally filtering buses by maximum distance from a given point.

Retrieves bus locations, optionally filtered by maximum distance from a given point.

This endpoint returns a list of bus locations after filtering out depoted buses. If the {\ttfamily max\+Distance}, {\ttfamily lat}, and {\ttfamily lon} parameters are provided, the list is further filtered by the specified maximum distance from the given latitude and longitude.

{\bfseries{Request Parameters\+:}}
\begin{DoxyItemize}
\item {\ttfamily max\+Distance} (Optional)\+: Maximum distance from the given point.
\item {\ttfamily lat} (Optional)\+: Latitude of the point for distance filtering.
\item {\ttfamily lon} (Optional)\+: Longitude of the point for distance filtering.
\end{DoxyItemize}

{\bfseries{Responses\+:}}
\begin{DoxyItemize}
\item {\ttfamily 200 OK}\+: Returns a list of filtered bus locations.
\item {\ttfamily 400 Bad Request}\+: Returns if {\ttfamily max\+Distance} is provided without {\ttfamily lat} or {\ttfamily lon}.
\end{DoxyItemize}
