\doxysubsection{Detailed Description}
Global class for storing predefined coordinates of bus depots.

This class initializes a list of {\ttfamily \doxylink{}{Bus\+Depot}} objects representing the coordinates of various bus depots. The depots\textquotesingle{} coordinates are manually added in the constructor.
