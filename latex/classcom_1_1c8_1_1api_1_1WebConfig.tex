Configures CORS mappings to allow cross-\/origin requests from any origin.

The configuration allows\+:
\begin{DoxyItemize}
\item Any origin pattern ({\ttfamily allowed\+Origin\+Patterns("{}\texorpdfstring{$\ast$}{*}"{})})
\item Only GET methods ({\ttfamily allowed\+Methods("{}\+GET"{})})
\item Any headers ({\ttfamily allowed\+Headers("{}\texorpdfstring{$\ast$}{*}"{})})
\item Credentials to be included in the request ({\ttfamily allow\+Credentials(true)})
\end{DoxyItemize}

{\bfseries{Note\+:}} This is a permissive CORS configuration intended for local development purposes. In production, it is recommended to restrict allowed origins to known domains.
