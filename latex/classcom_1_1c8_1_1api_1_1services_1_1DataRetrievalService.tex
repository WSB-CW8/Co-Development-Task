\doxysubsection{Detailed Description}
Service for retrieving bus and tram data from an external API.

This service fetches real-\/time data about buses and trams in Warsaw, Poland, from the specified public API endpoint. The data is returned as a parsed {\ttfamily \doxylink{}{Incoming\+Data}} object.
