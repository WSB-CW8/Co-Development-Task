\doxysubsection{Detailed Description}
Service for filtering bus locations by distance.

This service provides functionality to filter bus locations based on a given maximum distance from a specified latitude and longitude. It calculates the distance between the given coordinates and each bus\textquotesingle{}s location using the Haversine formula.
