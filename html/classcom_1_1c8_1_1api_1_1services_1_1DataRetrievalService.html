Detailed Description

Service for retrieving bus and tram data from an external API.

This service fetches real-time data about buses and trams in Warsaw, Poland, from the specified public API endpoint. The data is returned as a parsed IncomingData object.

inline

Fetches bus and tram data from the external API.

Sends a GET request to the external API endpoint to retrieve the latest bus and tram location data. The response is parsed into an IncomingData object.

Returns
An instance of IncomingData containing the bus and tram location data, or null if an error occurs.
