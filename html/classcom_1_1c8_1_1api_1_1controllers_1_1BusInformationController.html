Detailed Description

REST controller for retrieving information about bus locations.

This controller handles requests for fetching bus data, filtering depoted buses, and optionally filtering buses by maximum distance from a given point.

Retrieves bus locations, optionally filtered by maximum distance from a given point.

This endpoint returns a list of bus locations after filtering out depoted buses. If the maxDistance, lat, and lon parameters are provided, the list is further filtered by the specified maximum distance from the given latitude and longitude.

Request Parameters:

  • maxDistance (Optional): Maximum distance from the given point.
  • lat (Optional): Latitude of the point for distance filtering.
  • lon (Optional): Longitude of the point for distance filtering.

Responses:

  • 200 OK: Returns a list of filtered bus locations.
  • 400 Bad Request: Returns if maxDistance is provided without lat or lon.
Parameters
maxDistanceOptional maximum distance from the specified point.

A ResponseEntity containing the filtered list of bus locations or a bad request response.
