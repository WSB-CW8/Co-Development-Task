Detailed Description

Global class for storing predefined coordinates of bus depots.

This class initializes a list of BusDepot objects representing the coordinates of various bus depots. The depots' coordinates are manually added in the constructor.

Constructor that initializes the list of bus depots.

The constructor adds several hardcoded depots to the list, each with a name and bounding coordinates (latitude and longitude). Uncommented lines represent currently active depots.
